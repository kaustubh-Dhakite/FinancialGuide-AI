## 📌 Problem Statement

FinancialGuide AI is an AI-powered Digital Financial Literacy Assistant developed using **IBM Watsonx Orchestrate** and **IBM Granite**. The assistant helps users understand digital finance concepts, safely use online banking services, and make informed financial decisions through conversational AI and Retrieval-Augmented Generation (RAG).

---

## 🎯 Objectives

- Improve digital financial literacy.
- Educate users about UPI and digital payments.
- Raise awareness about online frauds and scams.
- Explain banking, loans, investments, and credit scores.
- Provide budgeting and personal finance guidance.
- Deliver reliable information using AI and trusted knowledge sources.

## 🏗 System Architecture

```
User
   │
   ▼
Watsonx Orchestrate
   │
   ▼
IBM Granite Model
   │
   ▼
Knowledge Base (RAG)
   │
   ▼
AI Generated Response
```

---

## 🔄 Workflow

1. User asks a financial question.
2. Watsonx Orchestrate processes the request.
3. Relevant information is retrieved from the knowledge base.
4. IBM Granite generates an accurate response.
5. User receives easy-to-understand financial guidance.
